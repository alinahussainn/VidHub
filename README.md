# 📸 VidHub
This is a  video-sharing platform that enables users to create accounts, upload and manage videos, interact via comments and likes, and offers both user and admin controls for managing content.

# 💫 Features
- user registration and authentication
-  video upload and streaming
-  video categorization
-  commenting and likes
-   personalized user dashboards
-   admin controls for content moderation and user management.
